# Software Requirements Specification (SRS) & System Testing

## Project Information

**Project Title:** BookBuddy - 전공책 중고거래 플랫폼

**Project Description:**  
대학생들은 매 학기 새로운 전공 서적을 구매해야 하는 경제적 부담을 안고 있으며, 사용한 전공책은 활용도가 낮아 처분에 어려움을 겪는다. 본 프로젝트는 같은 학교 학생들 간 전공책을 안전하게 거래할 수 있는 플랫폼을 제공하여 경제적 부담을 줄이고 자원 재활용을 촉진한다. 학교 인증을 통한 신뢰성 확보, 도서 상태 등급 시스템, 찜 기능, 거래 후기 시스템 등을 통해 사용자 간 안전한 거래 환경을 제공한다. 또한 관리자 기능을 통해 신고 처리 및 부적절한 게시글을 관리하여 건전한 커뮤니티를 유지한다.
